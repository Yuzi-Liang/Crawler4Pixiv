# Crawler4Pixiv
A Pixiv Crawler (Spider)

## Usage

Before you use, remember to do the following two steps:

1. Requirement (Python3)

- gzip

(Yes, gzip is the only required package)



2. Set config.py

```python
savePath = 'D:/pixiv/'
userID = ''
cookie = ''
```

Enter your Pixiv id, which can be found on your Pixiv homepage:
